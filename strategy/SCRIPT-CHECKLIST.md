  # The Master Script Checklist — all 3 videos in one gate

  Every short-form script must pass ALL of this before it ships. If any box fails, rewrite.
  Source: video 1 (algorithm), video 2 (hooks), video 3 (addictive storytelling). Ordered by how
  fast a script dies if it fails.

  ---

  ## GATE 0 — Avatar consistency (video 1: the #1 reach killer)
  The algorithm builds a fit-score from content signals. Mixed avatars = mixed sample = throttled.
  - [ ] **One avatar.** Reads to the algorithm as: *the tech-native person drowning in info who wants to stay sharp* (dev/researcher/founder). See AVATAR.md.
  - [ ] **No wrong-tribe signals.** NOT eco/planet, NOT touch-grass/wellness, NOT crypto-degen/trading, NOT anti-tech/conspiracy, NOT normie-consumer.
  - [ ] **Lane = flavor, not a new tribe.** AI/science/gaming/crypto/philosophy appear as *this person's* interests, framed for them (crypto = mechanism, not price; gaming = smart hobby; science = "wait-what").
  - [ ] **Self-test:** "If the algorithm only saw this, would it show it to a curious HN-reading dev?" If not → kill/reframe.

  ## GATE 1 — The Hook, first 1–2s (video 2: the 80/20)
  The hook's only job: opt-in via **topic clarity + on-target curiosity**. Kill all 4 mistakes:
  - [ ] **No Delay.** Topic/value in the FIRST sentence. No wind-up, no "so I was thinking…". Speed to value.
  - [ ] **No Confusion.** 6th-grade clarity. Short. ACTIVE voice. Only ONE way to interpret it.
  - [ ] **No Irrelevance.** Uses **"you/your"** and hits a pain they *already have* (being behind, drowning, noise, fried focus). Not "I/me" as the lead.
  - [ ] **No Disinterest.** Built-in **contrast** — A (what they believe/do now) vs B (the surprising alternative). Stated or implied.
  - [ ] Hook is ≤ ~2 sentences and could stand alone as the on-screen text.

  ## GATE 2 — The Addiction Loop (video 3: keeps them to the end)
  Dopamine = the *prediction* chemical. Keep an unresolved prediction running, no gaps. All 4 steps present, in order:
  - [ ] **STAKES** — a character to root for (usually "you" or "me") + something at risk + **urgency/ticking clock**. Personally relevant to the avatar (not life-or-death, just *matters to them*).
  - [ ] **BIG QUESTION** — gives *enough specifics to make them PREDICT* (not a vague "you won't believe"). Loaded EARLY so dopamine drips longer. There's a clear question their brain wants to solve.
  - [ ] **HEAD FAKE** — the reveal BREAKS their prediction but is **logical in hindsight** (the clues were there). *Threadverse is the head fake* — the satisfying "ohhh, it wasn't luck/willpower, it was the product" answer. Not random, not a non-sequitur.
  - [ ] **REHOOK** — closes that loop AND opens a new one **in the same breath** (connective phrase: "which would've been great, except…", "but here's the problem…", "and that's when I realized…", "which is exactly why…"). ZERO dead air before the CTA.

  ## GATE 3 — Engagement drivers (video 3 + hooks: get comments/shares)
  At least ONE, ideally two, without feeling forced:
  - [ ] A **hard, slightly contrarian stance** (contrarian > hedging — drives comments).
  - [ ] **Emotion** — surprise / mild outrage / "wait, really" / dry delight.
  - [ ] A **cult-loved / opinion-triggering entity** named where natural (a specific AI model, studio, chain, thinker) — people jump in when they already have a take.

  ## GATE 4 — Native, NOT an ad (the big one — obvious marketing kills the head fake + the algorithm files it as "promo")
  The whole trap: if the viewer sees "this is an ad for a feed app" by second 2, there's no prediction
  error (no dopamine) AND the algorithm samples "promotional content" → throttled. Fix: the story/value
  is the point; the product is incidental.
  - [ ] **The delete test.** If you removed every Threadverse mention, is this STILL a genuinely good post (a real story or real value)? If no → it's an ad, rewrite.
  - [ ] **Story-first (~60%)** or **value-first (~40%)**:
    - Story-first: a real, complete micro-story with a concrete character in a concrete moment. Threadverse is *how it happened*, mentioned once.
    - Value-first: teach the avatar something genuinely novel/useful (mechanism, finding, contrarian take). Threadverse is just *where I found it*.
  - [ ] **Product is the SETTING, not the SUBJECT.** Named at most ONCE, casually ("…my Threadverse feed had it"), and shown in frame (B-roll/screen-record). Never "here's an app that…".
  - [ ] **The head fake is a REAL surprise**, not "…and the answer is: buy this." The twist is in the *story/idea*; the product is ambient, not the punchline.
  - [ ] **CTA is optional** — not every script ends with "threadverse.ai". When present, it's soft and earned; often the link lives in bio/caption instead.
  - [ ] First line reads like a peer telling you something interesting — zero pitch smell.

  ## GATE 5 — Craft / production-ready
  - [ ] Runtime 20–40s (short-form).
  - [ ] Every beat has a `[V: …]` visual cue the user can generate.
  - [ ] Consistent voice: casual, sharp, dry, faintly contrarian; talking to a peer; "you/your".
  - [ ] Ends on the rehook→CTA with no gap. No "anyway" / no trailing off.

  ---

  ## One-line pass test (say yes to all)
  > Same avatar? · Hook clear + contrast in line 1? · Stakes→Big-Q→Head-fake→Rehook all present, no gap? ·
  > One contrarian/emotional/entity driver? · Product is the head fake, CTA earned? · Every beat has a visual?

  If all yes → ship. If any no → the checklist tells you exactly which gate to rewrite.
